# Paggy

Pagination engine for Rails 7 w Minitest

## Usage

## Installation

Add this line to your application's Gemfile:

```ruby
gem "paggy"
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install paggy
```
